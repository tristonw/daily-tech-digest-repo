- **Subquadratic（LLM 数学瓶颈）**：如果证据被独立复现，可能形成新一轮基座价格战，长上下文 SaaS 单位经济学被重写。值得作为"6 个月内技术雷达高优先级跟踪项"。

## 🏢 企业风控·保险场景借鉴

1. **供应链审查必须升级到"运行时验证"**。1 万木马仓库事件意味着传统的 Star/Fork/Readme 信号失效。建议立刻：① 引入 SBOM 强制生成；② 把 GitHub 拉包白名单化、签名化；③ CI 引入二进制扫描（不只看源码）；④ 内部模板与脚手架走签名分发。对保险公司而言，对客户提供的 IT 责任险、网络安全保险，**应将"是否使用 SBOM + 是否签名拉包"作为承保条件与折扣项**。
2. **TimesFM 作为精算辅助工具的 PoC**。一周内启动一个内部 PoC：用 TimesFM 对历史理赔频次、保费现金流、销售曲线做零样本预测，与现有精算模型比对。即使最终不替换，也能形成"基础线 + 现有模型差异图"，为模型审计加一条独立视角。
3. **Agent 在风控/尽调/客服的真正落地路径是"Skills + 压缩 + 记忆"**。基于 obra/superpowers 思路把内部 SOP 沉淀为 Skill；用 headroom 思路压缩工具输出（如长篇尽调资料、客户上传文件）；用 codebase-memory-mcp 思路给"客户档案 + 历史交互"做持久化记忆。这三件事一起做，才能把单次 Agent 任务的成本从美元级降到分钱级，进入大规模放量阈值。
4. **VMware / Atlassian / Adobe / Postman 替代的"统一窗口期"**。makeplane/plane、penpot、Kong/insomnia、Ubiquiti 企业 NAS——同一周内多个赛道的开源/低价替代同时被点亮。建议 IT 与 CIO 团队启动"五年合约重谈包"，对所有 8 位数美元年费 SaaS 做一次系统替代评估。
5. **GDPR 执法严化 → 保险产品机会**。Elkjop 1.8M 欧元罚单提示，欧盟"暗模式 / 强制同意"罚款从警告进入实质执法。GDPR / 隐私违规赔偿险、Cookie 合规咨询险种的可保性与定价模型可以重新评估。
6. **BCI 临床长期使用 → 新险种建模**。Casey Harrell 案是 BCI 商业临床长期化的标志。需要提前布局：① 植入式医疗设备故障险；② BCI 软件升级失败 / 系统不可用造成的恶化责任险；③ 医生 / 厂商 / 软件供应商的复合责任分担条款。
7. **Boston Dynamics 估值回归现金流逻辑 → 工业机器人保险**。3.25 亿美元的交易价提示具身智能下一站是工业与车厂，而不是消费。工业机器人责任险、机器人协作工伤险、机器人产品召回险将是 2026-2027 的细分增长点。
