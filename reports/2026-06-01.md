## 📊 趋势分析

1. **Agent 栈从工具走向"包管理器 + 路由 + 记忆"三件套。** GitHub 这边 [anthropics/skills](https://github.com/anthropics/skills)、[ECC](https://github.com/affaan-m/ECC)、[supermemory](https://github.com/supermemoryai/supermemory) 全部冲进高位，资本面 [OpenRouter B 轮](https://openrouter.ai/announcements/series-b) 把"模型路由"封为基础设施层。CTO 们在 2026 下半年要选型的，不再只是一个模型或一个 IDE，而是 Skills 来源、Memory 后端、Router 通道这三层的组合。
2. **「永久授权」这个商业承诺，正在被云时代彻底重写。** Microsoft 在更新里把买断版 Office 改成只读，引发 HN 头条；与此同时 [Downdetector + Speedtest 以 12 亿美元被 Accenture 收购](https://www.theverge.com/tech/889234/downdetector-ookla-speedtest-sold-accenture)——前者把"软件资产"变成了短租，后者把"运行时遥测"卖成了金矿。两端一起讲一个故事：软件的价值正在从「卖出去的那一刻」转移到「持续被使用、被观测的那个过程」。企业资产负债表里的软件项，得重新算寿命与折旧。
3. **AI 落地的真正瓶颈，从"模型够不够强"切换到"组织够不够新"。** MIT Technology Review 给出 85%/76% 的对比，HN 第二条 [《Domain expertise is the moat》](https://www.brethorsting.com/blog/2026/05/domain-expertise-has-always-been-the-real-moat/) 给出原因。模型差距在缩小，行业 know-how 和组织流程改造的难度被放大。配上 [AI Hype Index 中毕业生集体嘘 AI](https://www.technologyreview.com/2026/05/28/1138053/the-ai-hype-index-ai-gets-booed-in-graduation-season/) 这条情绪信号，2026 年下半年要赢的，是「会用行业知识喂养 Agent」的团队，不是「最先买到模型」的团队。

## 💼 商业化价值

- **模型路由层（OpenRouter 类）**：[1.13 亿美元 B 轮](https://openrouter.ai/announcements/series-b) 已经把估值锚定。盈利路径清晰——按 token 抽佣、按延迟与可靠性 SLA 收溢价、企业版加审计与脱敏。对企业是天然的「多供应商」管控点，可以避开单一模型断供风险。
- **Memory / Skills 包管理（supermemory、anthropics/skills、ECC）**：商业模式仍在长。短期靠开源造势 + 托管版收费；长期可能演化为 Agent 时代的 npm/pypi，市场天花板可对标软件包注册中心 + APM 可观测性平台的组合。
- **数据预处理与爬取（[markitdown](https://github.com/microsoft/markitdown)、[Scrapling](https://github.com/D4Vinci/Scrapling)）**：RAG 与 Agent 商业化都需要海量"喂料"。预处理这块更适合做成 SaaS 或私有部署组件，按文档量计费。
- **本地 TTS / 边缘 AI（[VoxCPM](https://github.com/OpenBMB/VoxCPM)、[project-nomad](https://github.com/Crosstalk-Solutions/project-nomad)）**：高质量本地推理逐步把成本下探到端侧。客服、智能硬件、应急指挥这些场景里，"离线 + 数据不出域"会变成保险与监管合规企业的硬需求。
- **被动感知（[RuView](https://github.com/ruvnet/RuView)）**：WiFi 信号做生命体征监测、占用检测，绕开摄像头隐私门槛。可以切入仓储管理、商业地产人流量、长护险居家照护这些 B 端市场，毛利与隐私合规天然友好。
- **遥测数据资产化（[Downdetector / Speedtest](https://www.theverge.com/tech/889234/downdetector-ookla-speedtest-sold-accenture)）**：Accenture 12 亿美元的报价，给所有「掌握全行业可用性、性能数据」的公司一份估值参照。对企业内部，这也是把 APM、CMDB、可用性数据"商品化"的暗号。
